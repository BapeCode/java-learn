## Exercice 1 : La présentation (Variables & Affichage)
Ton but est de créer un programme qui stocke des informations sur un personnage (toi ou un personnage fictif) dans des variables, puis qui affiche une phrase complète.

Les consignes :

Dans la méthode main, déclare 4 variables avec les bons types :

- Une variable pour le prénom (type String).

- Une variable pour l'âge (type int).

- Une variable pour la taille en mètres (type double, ex: 1.75).

- Une variable pour savoir si la personne aime le Java (type boolean, met true ou false).

Utilise ```System.out.println(...)``` pour afficher une phrase qui combine ces variables.

Ce que la console doit afficher (exemple) :

```Salut, je m'appelle Thomas. J'ai 25 ans et je mesure 1.8m. J'aime Java : true```

Aide-mémoire pour la syntaxe :

Pour coller du texte et une variable ensemble, on utilise le signe +.

- Exemple : ```System.out.println("J'ai " + age + " ans");```

N'oublie pas le point-virgule ; à la fin de chaque ligne !
